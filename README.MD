## Getting Started

### Requirements

-   Node 18 or later
-   npm 9 or later

### Run the Project Locally

-   npm install
-   npm run build
-   npm start

This will open a browser and your storefront will be running on http://localhost:3000

### Navigate my custom component

http://localhost:3000/my-new-route/
